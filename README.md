# TakeYourMedicineBot — Лендинг

Лендинг для Telegram-бота, который помогает не забывать принимать лекарства вовремя.

## О проекте

**TakeYourMedicineBot** — Telegram-бот, который отправляет напоминания о приёме лекарств. Этот репозиторий содержит лендинг-страницу для бота.

## Технологии

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Локальная разработка

```sh
# Клонировать репозиторий
git clone <YOUR_GIT_URL>

# Перейти в папку проекта
cd <YOUR_PROJECT_NAME>

# Установить зависимости
npm i

# Запустить сервер разработки
npm run dev
```
