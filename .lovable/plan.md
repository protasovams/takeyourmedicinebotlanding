# 💊 PillReminder — Лендинг для Telegram-бота

## Обзор

Одностраничный адаптивный лендинг для Telegram-бота "Напоминание о лекарствах" с фирменной цветовой схемой в стиле Telegram, плавными анимациями при скролле и полностью адаптивным дизайном.

---

## Секции лендинга

### 1. Фиксированный хедер

- Логотип "💊 Выпей лекарства" слева
- Горизонтальное меню навигации: Главная, Возможности, Как работает, FAQ, Контакты
- Кнопка "Открыть бота" (CTA)
- На мобильных — бургер-меню с выезжающей панелью
- Прозрачный фон, становится белым при скролле

### 2. Hero-секция (полный экран)

- Крупный заголовок: "Телеграм-бот, который вовремя напомнит принять лекарства"
- Подзаголовок и CTA-кнопка "Начать бесплатно"
- Мокап телефона с интерфейсом Telegram-чата (плейсхолдер)

### 3. Проблема и решение

- Заголовок "Забываешь лекарства?"
- Описание преимуществ бота
- 3 карточки: "Без приложений", "Автоматически", "Просто" — с иконками и описаниями
- На мобильных карточки в столбик

### 4. Функциональность

- Заголовок "Что умеет бот"
- 6 карточек сеткой 2×3 (на мобильных — столбик):
  - Список лекарств, Напоминания, Отложить, Статистика, Мотивация, Обратная связь
- Каждая карточка: иконка + заголовок + краткое описание

### 5. Как работает

- Заголовок "3 простых шага"
- 3 пронумерованные карточки с описанием шагов: Открой бота → Настрой лекарства → Получай напоминания
- Плейсхолдеры для скриншотов Telegram

### 6. FAQ

- Заголовок "Часто задаваемые вопросы"
- 4 аккордеона с вопросами и ответами о боте

### 7. Футер

- Индиго фон, белый текст
- Логотип, ссылки (Обратная связь, Политика, @username)
- Кнопка "Открыть бота"
- Копирайт с дисклеймером

## Дизайн и UX

- **Цвета**: Светлая схема в синих тонах. Избегай градиентов
  **Фон страницы:**
  - #F4F9FF — очень светлый холодный голубой
  **Основные блоки / карточки:**
  - #FFFFFF
  - Бордер: #E3EEF9
  **Основной цвет (бренд / заголовки / активные элементы):**
  - #229ED9
  - Hover: #1C8AC0
  **CTA-кнопка (главная):**
  - Фон: #229ED9
  - Текст: #FFFFFF
  - Hover: #1A7FB3
  - Лёгкая тень: rgba(34, 158, 217, 0.25)
  **Вторичный акцент (иконки, активные состояния):**
  - #38BDF8
  **Текст:**
  - Основной: #1F2937
  - Вторичный: #64748B
  - Disabled: #94A3B8
  **Успех:**
  - #16A34A
  **Ошибка:**
  - #DC2626
- **Типографика**: Inter, заголовки bold 48/36/24px, текст 18/16px
- **Анимации**: fade-in up при скролле (Intersection Observer), hover scale 1.05 на кнопках
- **Адаптивность**: мобильные — карточки в столбик, бургер-меню; десктоп — горизонтальная сетка и меню
- Плавный скролл, секции с большими отступами
